# 社区行为准则 — PromiseLink

## 我们的承诺

为营造开放友好的社区环境，所有成员、贡献者和维护者承诺：

- **包容性**：无论经验水平、性别、性取向、残疾、外貌、身体尺寸、种族、民族、宗教信仰，均受尊重
- **建设性**：以技术事实为依据，避免人身攻击
- **协作性**：乐于帮助新手，分享知识
- **尊重性**：尊重不同观点，接受建设性批评

## 我们的标准

### 正面行为

- 使用友好包容的语言
- 尊重不同观点和经验
- 优雅地接受建设性批评
- 关注社区整体利益
- 对其他社区成员表示同理心

### 不可接受的行为

- 使用性暗示的语言或图像
- 人身攻击、恶意挑衅、辱骂
- 公开或私下骚扰他人
- 未经许可泄露他人私人信息
- 其他违反职业道德的行为

## 维护者责任

项目维护者有权：

- 删除违反本准则的评论、提交、代码、wiki 编辑、issue 和 PR
- 暂时或永久封禁违反者
- 拒绝不符合项目利益的贡献

## 适用范围

本准则适用于所有项目空间（GitHub 仓库、Issues、PR、Wiki、讨论区）以及公共空间中个人代表项目时的行为。

## 执行

违反本准则的行为可向 **conduct@promiselink.app** 举报。所有举报将被审查并妥善处理。

所有维护者和贡献者必须遵守本准则。不遵守可能导致暂时或永久封禁。

## 出处

本行为准则改编自 [Contributor Covenant](https://www.contributor-covenant.org/) v2.1。
